This is a placeholder. The repository structure, license, and documentation are in place. The tool itself is not yet implemented.

Planned tool
Browser-based calculator for noise barrier attenuation, using ISO 9613-2 Maekawa diffraction. Single-source, single-barrier, single-receiver or grid. Shares its physics engine with acmap.

Development is scheduled after the integration of the companion acmap tool into the ST-LINE website. See the project README for the full roadmap and motivation.

If you want to be notified when the first working version ships, watch the GitHub repository.
